# ジェットコースターメイカー

スマホ縦持ち向けのブラウザゲームMVPです。  
2本のラインを描くと、そこから3Dジェットコースター風のコースを生成し、一人称視点で自動走行します。

## ファイル構成

- `index.html` : 画面構成
- `style.css` : スマホ向けの見た目
- `main.js` : 入力、コース生成、走行、ノリコさん、スコア計算

## ローカル起動

`index.html` を直接開いても動きますが、簡単なローカルサーバー経由がおすすめです。

```bash
cd /Users/assmagic/Desktop/ジェットコースター
python3 -m http.server 8000
```

ブラウザで以下を開いてください。

- PC: `http://127.0.0.1:8000`
- スマホ: 同じWi-Fi内のPCのIPアドレスを使って `http://<PCのIP>:8000`

## 実装メモ

- STARTからGOALまで左から右へ描く仕様です
- 横からの線を高さ `Y`、上からの線を左右位置 `X` に変換しています
- `Z` は進行方向として一定で増加します
- 入力線はリサンプリングと平滑化を行ってから3Dコース化しています
- 走行中は速度、勾配、カーブ強度から恐怖値を作り、ノリコさんの表情を変えています
